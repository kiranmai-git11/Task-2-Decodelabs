## 📌 About This Task
This task simulates real IoT sensor data collection
using JavaScript — no hardware needed.
It generates Temperature, Humidity, and Motion
sensor readings and logs them with timestamps.

---

## ✅ What I Built
- Live sensor data simulation (Temperature, Humidity, Motion)
- Start / Stop / Clear controls
- Adjustable update interval (1s, 2s, 5s)
- 4 scenario modes to test different conditions
- Data log table with timestamps and status
- Min/Max statistics tracking
- CSV export to download all logged data

---

## 🔧 Sensors Simulated
| Sensor | Unit | Normal Range | Alert Level |
|--------|------|-------------|-------------|
| Temperature | °C | 21–27°C | > 38°C |
| Humidity | % | 45–65% | > 85% |
| Motion | detected/none | 20% chance | Any detection |

---

## 🎮 Scenario Modes
| Mode | Description |
|------|-------------|
| Normal | Everyday indoor conditions |
| Heat Wave | High temperature simulation |
| High Humidity | Humid environment simulation |
| Motion Active | Frequent motion detection |

---

## ▶️ How to Run
1. Download `task2_sensor_simulation.html`
2. Open it in any browser
3. Click **Start** to begin simulation
4. Change scenario from dropdown
5. Click **Export CSV** to download data.
